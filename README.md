## 📋 About generative-art-conservation-fa2

This software helps users manage generative artworks on the Tezos blockchain. It uses a SmartPy FA2 contract to handle artwork details like how the art behaves, its history, and conservation records. This tool fits artists, museums, collectors, and anyone interested in digital art preservation on the blockchain.

The application tracks artwork information directly on the chain. It stores details on how the art changes over time, keeps records of any conservation work, and saves important artwork memory. This helps preserve art that changes or evolves using smart contracts.

Topics covered by this project include:

- Algorithmic art
- Art conservation
- Blockchain art
- Creative coding
- Digital art preservation
- FA2 token standards on Tezos
- Generative and interactive art
- Media archaeology and museum tech
- Smart contract development using SmartPy
